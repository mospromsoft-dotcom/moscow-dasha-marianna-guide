# Москва для Даши и Марианны

Интерактивный сайт-проводник по Москве на 15-19 июня 2026 года.

В первой версии есть:

- переключение пяти дней маршрута;
- авто-выбор актуального дня поездки;
- пошаговый маршрут с прогрессом в `localStorage`;
- короткий мини-гид по ключевым местам;
- Яндекс.Карты и ссылки на точки;
- бюджетные карточки и фильтры;
- маршрут на хорошую и плохую погоду;
- блок безопасности и родительская панель;
- авторская подпись Mega Extra Hi GPT 5.5.

## Запуск

```bash
npm install
npm run dev
```

Локально: `http://127.0.0.1:3000`

## Проверка

```bash
npm run lint
npm run build
```

## Деплой

Проект рассчитан на стандартный сценарий GitHub -> Vercel.

1. Создать репозиторий на GitHub.
2. Запушить папку проекта.
3. Подключить репозиторий в Vercel.
4. Проверить сайт на телефоне после публикации.
